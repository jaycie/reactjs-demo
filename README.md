# reactjs-demo


## A demo use reactjs,react-router,fontAwesome

#### 使用 reactjs,react-router,fontAwesome等组合，适合初学者组织文件目录，fetch远程数据学习

#### 下载组件

```
$ git clone https://github.com/jaycie/reactjs-demo/
```

#### 运行

```

$ cd reactjs-demo
$ npm install
$ npm start
```

#### 部署到开发环境

```

$ npm run build
```
